# Lists Application

## Overview
This project is a simple lists application that allows users to manage a collection of items. It provides functionalities to add, remove, and retrieve items from a list.

## Project Structure
```
lists
├── src
│   ├── index.ts
│   ├── components
│   │   └── List.ts
│   ├── utils
│   │   └── helpers.ts
│   └── types
│       └── index.ts
├── package.json
├── tsconfig.json
└── README.md
```

## Installation
To get started with the project, clone the repository and install the dependencies:

```bash
git clone <repository-url>
cd lists
npm install
```

## Usage
To run the application, use the following command:

```bash
npm start
```

## Features
- Add items to the list
- Remove items from the list
- Retrieve all items in the list
